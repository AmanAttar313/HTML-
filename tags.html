HTML was invented by Tim Berners-Lee, a physicist at CERN, in 1990. He aimed to create a simple way to share and access documents over the Internet.
Since its inception, HTML has evolved significantly, becoming the foundation of web development.

Features of HTML

HTML employs tags to structure content, defining elements like headings, paragraphs, lists, and links, facilitating document organization and readability for web browsers. Compatible with various devices and operating systems, HTML ensures consistent rendering and accessibility across different platforms.
